# CONTENTS:

  emcee             ... Copy of the python EMCEE MCMC sampler.
  mpfit.py          ... Port of Levenberg-Marquardt fitting routine.
  normalize.py      ... Copy of image transfer functions from APLpy.
  util_FITS.py      ... Utility routines to operate on FITS format files.
  util_misc.py      ... Routines for working with Faraday spectra & cubes.
  util_plotFITS.py  ... Routines for plotting fits images using matplotlib.
  util_plotTk.py    ... Routines for plotting Faraday spectra.
  util_rec.py       ... Routines for operating on python record arrays.
  util_RM.py        ... Routines for performing RM-synthesis and CLEAN.
